而正巧A一直单身，就在我以为A就要脱离我们广大的单身狗行列的时候。

世界匆匆忙忙，但她在乎你长得舒不舒畅
2023-02-01 2 min read # 情感

我发现有个能主动过问你情绪的人太美好了。

前天晚上我情绪很差，于是分享了一首歌，配文“好累”。说实话那条动态蛮矫情的，所以我也没想到会有朋友来找我。

结果过了几分钟，对话框就亮了。我一个好友直截了当的问我，怎么了，有什么不开心？我赶紧说没啥，都是小事，不过我正好可以分享点八卦给你。

本以为我转移话题的那一套很管用，结果她突然正经起来，发来一段话：“你不用怕麻烦我，就算是小事我也愿意听。”

真的，就那一刻，心里的委屈一下涌上来了。然后我就絮絮叨叨跟她说了好多，都是屁大点的烦恼，她也认真回应着。

看得出来她抗拒我的肢体接触，哪怕是安抚她这会儿也不想，她大声吼着让我滚开，我摆着手说好的好的，就往后退。她站在鞋柜旁边，我蹲在地上捡玻璃渣子。余光瞥见她光着脚，刚想给她拿鞋子的时候她突然转身开门出去了。

我在恋爱前期还挺有安全感的。

因为我知道我不差，他们喜欢我很正常，当然如果他们遇到更好的我也觉得无所谓，因为我也能遇到更好的。

可一旦用心了，感受到了对方的真诚和喜欢，就好怕这份爱会因为自己哪里不够好而消失啊，所以每次我动心之后都感觉战战兢兢的。

刚在一起时对方不给我发消息我也不在乎，投入之后对方超过三个小时不给我发消息我就会想他在干嘛啊，为什么不给我发消息，为什么不主动联系我，不会在和别人聊天吧，是我不够好吗，不会是不喜欢我了吧。

因为我害怕，我不想再被谁毫不在意的丢掉
2022-02-25 5 min read # 情感

我常听人说男生的喜欢是减分制的。

你对他的喜欢越多，他对你的喜欢越少，你越是对他好，他就越嫌你无聊。

但我不行，我做不到这样。我太容易对一个人好了，根本控制不住的那种。当我感受到别人给了我三分甜的时候，我会很想努力把剩下的七分甜都补给对方。所以常常在一份感情里，对方才刚有一点喜欢我的时候，我就已经在心里很喜欢很喜欢对方了。

我以前特别喜欢一个男孩子，有多喜欢呢，是那种我们隔着上千公里的距离，为了想见他一面，我就会立刻收拾行李买机票的那种喜欢。

对方喜欢我吗？也是喜欢过的。

追我的那会儿，他每天至少给我打五个电话。早上九点多准时开始问我起床了吗，吃饭了吗，在干嘛。中午的时候又问我吃饭了嘛，要不要休息一会啊？到了晚上就开始想话题找我闲聊，有时候我不回信息他也不怪我，反而睡觉之前还和我说晚安。

今年过年的时候，他来我家拜年了
2022-02-21 3 min read # 情感

我看到这样一个故事，女生早恋被她爸爸知道后男生勇敢的站出来，两人毕业后修成正果的结局。

原来，大家的故事这么相似又这么不同。我也是高中早恋，晚自习关灯后我们在教室牵手被教导主任看到了。灯亮的瞬间我们慌乱到不行，我爸对于早恋是零容忍的。我用哀求的目光看向主任，他什么都没说，只是让我们赶紧走。回家后，看着爸妈对我失望的眼神，我觉得特别内疚。

那天过后，我删了他的联系方式，我们整个学期再也没有说一句话，再也没有聊过一句天。就好像从来都不认识一样。只是，我天天都能从我的空间访客里看到他，每次他用愧疚的眼神看向我，我都会执拗的转过头。他拖人写给我的纸条，每天书桌里的那瓶阿萨姆，全被我丢进了垃圾桶。

要珍惜那个能在你身边睡个好觉的人
2021-12-06 1 min read # 情感

我今天看到一个小知识。

老虎是不会嗷呜嗷呜叫的。所以为了表示开心，他们会眯上眼睛，或者直接闭上眼睛。

这时候，他们的视力会变得模糊，防御能力也会大大降低，也就代表，它感到非常的舒服并拥有相当的安全感。

我想说，爱一个人也是一样的啊。

当一个人睡在你左侧，如果他辗转反侧，那或许他正忐忑地喜欢着你。

但如果一个人在你身边，能够闭上双眼，睡得香沉踏实，完全不担心周遭变化，那就是ta用爱和信任在向你完全的投诚。

能够长长久久的亲密关系一定不是计算，更不是算计
2021-02-09 4 min read # 情感

我看见一条私信，是个姐姐，她说自己好像活了快四十岁了，最近才知道怎么爱人。

“年轻那会儿一心想找称心如意的男孩儿结婚，要求学历，要求长相，要求三观合拍，要求个性相契，心里似乎一直有个打分表，每一段儿恋爱都不由自主的边处边评估。大方慷慨，加一分，说话没分寸，扣一分，准备惊喜，加一分，不够体贴，扣一分，加加减减，上上下下，分数一旦不及格就判定为不是真爱，果断抽身寻找下一任。”

姐姐说，我现在才意识到真爱根本不是这样计算出来的。年龄的增长根本就没有为我带来什么人生经验，我的小半生，都在错误的标准里打转。

就想到我室友，二十出头的小姑娘，一段儿恋爱谈了十年，从初二到研一，十年长跑，八年异地，俩人仍然如胶似漆，视频一打就是俩小时起步，每每提及男友都一脸柔情蜜意。

某次聊天儿，我问她怎么做到经营一段感情十年不变质的，中间不会有矛盾冲突想要分手的时刻吗？
室友楞了一下摇摇头：“从来没想过。”
